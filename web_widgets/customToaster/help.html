Use this widget to:

  • Handle toastrs (short notifications)

For Instance to let users know when a successful rest call is done:

  • Create a new variable of type json named toastrs with value 

    []

  • Create a new variable of type javascript expression with the following code. 

    //add your condition to trigger the notification here
//if( ... ){
    var toast = {};
//Style that will be used:
// primary, success, info, warning, danger
    toast.type = "success";
 // Title of you notification
    toast.title = "Your call has been successful";
// Text you would like to show in the notification
    toast.body = "";
// Milliseconds that the toastr will be displayed
    toast.timeout = 3000;
// Push this toast into collection to publish
    $data.toastrs.push(toast);
// If you added a condition, uncomment following line
//}

  • Bind the toastrs variable of your form to the property toastrs
